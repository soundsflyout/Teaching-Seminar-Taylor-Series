Taylor Approximations of cos(x)#

Below is a graph of \(\cos(x)\) compared to its Taylor approximations. The formula for the approximations are

\[ p_{2n}(x) := \sum_{k=0}^n \frac{(-1)^k}{(2k)!}x^{2k} \;.\]

Here we only care about the even degrees since \(\cos^{(n)}(0) = 0\) when \(n\) is an odd number. In blue is the graph of \(\cos(x)\) and in red is the graph of \(p_n(x)\). Change the degree to see what happens!

Hide code cell source

 
taylor_approximate_cos = lambda degree, x: sum([((-1)**(k//2))*x**k/math.factorial(k) for k in range(0,degree+1,2)])


x =np.linspace(-200,200,10000)
cosine =np.cos(x)
cosine_approx = taylor_approximate_cos(4,x)

cos_approx = ColumnDataSource(data=dict(x=x, y =cosine_approx))
cos_true = ColumnDataSource(data=dict(x=x, y=cosine))

cos_plot = figure(title="Taylor Approximation of cos(x)", 
              tools="crosshair,pan,reset,save",
              x_range=[-6,6], y_range=[-5,5])

cos_plot.line('x', 'y', source=cos_approx, line_width=3, line_alpha=0.8, color = "red")
cos_plot.line('x','y', source=cos_true, line_width=3, line_alpha=0.8, color ="blue")

cos_degree = Slider(title="degree", value=4, start=0, end=20, step=2)

cos_callback = CustomJS(args=dict(source=cos_approx), code=""" 
    const d = cb_obj.value
    const x = source.data.x

    function cos_approx(d,x) {
        let soln = 0;
        let factorial = 1;
        for (let i=0; i <=d; i++) {
            if (i != 0) {
                factorial = factorial*i;
            }
            if (i % 2 == 0) { 
                soln += Math.pow(-1,i / 2)*Math.pow(x, i)/factorial ;
                }
            }
        return soln;
    }

    const y = Array.from(x, (x) => cos_approx(d,x))
    source.data = { x, y}
""")

cos_degree.js_on_change("value", cos_callback)
show(column(cos_degree, cos_plot))

Taylor Approximation of exp(x)#

Next we will check out the Taylor approximations for \(\exp(x)\). Since \(\exp^{(n)}(0) = \exp(0) = 1\) for all \(n\), we get the Taylor polynomials

\[ p_n(x) = \sum_{k=0}^n \frac{1}{k!}x^k \;. \]

That each term grows as the inverse of a factorial means that the terms converge fairly quickly.

taylor_approximate_exp = lambda degree, x: sum([x**k/math.factorial(k) for k in range(0, degree+1)])

Taylor Approximation of log(1+x)#

For the Taylor approximation of log(1+x), notice that the Taylor approximations will not converge for \(x > 1\). The Taylor polynomials are

\[ p_n(x) = \sum_{k=1}^n \frac{(-1)^{k+1}}{k} x^k \;. \]

taylor_approximate_log = lambda degree, x: sum([((-1)**(k+1))*((x-1)**k)/k for k in range(1,degree+1)])


x =np.linspace(0.001,20,1000)
logarithm =np.log(x)
logarithm_approx = taylor_approximate_log(1,x)

log_approx = ColumnDataSource(data=dict(x=x, y =logarithm_approx))
log_true = ColumnDataSource(data=dict(x=x, y=logarithm))

log_plot = figure(title="Taylor Approximation of log(1+x)", 
              tools="crosshair,pan,reset,save",
              x_range=[-1,6], y_range=[-5,5])

log_plot.line('x', 'y', source=log_approx, line_width=3, line_alpha=0.8, color = "red")
log_plot.line('x','y', source=log_true, line_width=3, line_alpha=0.8, color ="blue")

log_degree = Slider(title="degree", value=0, start=0, end=20, step=1)

log_callback = CustomJS(args=dict(source=log_approx), code=""" 
    const d = cb_obj.value
    const x = source.data.x

    function log_approx(d,x) {
        let soln = 0;
        for (let i=1; i <=d; i++) {
            soln += Math.pow(-1,i+1)*Math.pow(x-1, i)/i ;
        }
        return soln;
    }

    const y = Array.from(x, (x) => log_approx(d,x))
    source.data = { x, y}
""")

log_degree.js_on_change("value", log_callback)
show(column(log_degree, log_plot))
